# My Wine Notes Support

For help with My Wine Notes or for a privacy question, email
[joeycato+mywinenotes@gmail.com](mailto:joeycato+mywinenotes@gmail.com).

When contacting support, include your iOS version, app version, and a short
description of what happened. Please do not email sensitive personal
information or attach private tasting notes or photos unless they are necessary
to explain the problem.

For a non-sensitive bug report or feature request, you can also
[open a GitHub issue](https://github.com/joeycato/wine-notes-ios-support/issues/new).
GitHub issues are public.

[Read the Privacy Policy](https://joeycato.github.io/wine-notes-ios-support/privacy/).
